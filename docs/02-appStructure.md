# 02 - Estructura de la App con Respecto al Diseño

Este documento explica cómo la estructura de la aplicación se alinea con el diseño propuesto.

## Componentes Principales

- **Header**: Muestra el logo y título.
- **NeonLight**: Barra de neón como separador visual.
- **Recorder**: Componente principal para grabación de audio.
- **RgbContext**: Componente que da el color de la animacion rgb a los componentes
