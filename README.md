# PGL-SoundRecorder

PGL-SoundRecorder es una aplicación móvil desarrollada como practica de DAM. Hecha con React Native y Expo. Esta app permite grabar, reproducir y gestionar archivos de audio de manera intuitiva. Diseñada para usuarios que necesitan una herramienta de grabacion un poco personalizada como para "gamers" o "frikis" que le guste el RGB (RAINBOW GOES BRRRRR).
